# Run Locally

Clone the project

```bash
  git clone https://github.com/dimasrdika/KittyNest-startup.git
```

Go to the project directory

```bash
  cd KittyNest-startup
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
